## You can change Setting from StreamingAssets
```
<?xml version="1.0" encoding="utf-8"?>
<URGData>
  <URGStep>
    <StartStep>0</StartStep>
    <EndStep>1080</EndStep>
    <StepCount360>1400</StepCount360>
  </URGStep>
</URGData>
```
